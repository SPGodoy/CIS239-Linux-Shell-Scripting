# CIS239-Linux-Shell-Scripting

Course repository for CIS239 covering Linux Bourne shell scripting.

This repo includes:
- 9 assignments 
- Practice scripts and examples
- A capstone script combining core concepts

## Topics Covered
- Variables
- Wildcards
- Loops
- Test conditions
- Case statements
- Functions and exit codes

Scripts were written as part of coursework and show progression in shell scripting skills.
